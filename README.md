# SupabaseTutorialUsingFlutter
+ Create an account for yourself on supabase and make your own database which means changing the url and anonKey in the code. 

+ Be attentive to what your flutter project is called to change certain imports: 'package:my_app/note.dart' (your project may not be called my_app for example).

+ Also make sure that supabase is connected by running this in the terminal before you start anything: flutter pub add supabase_flutter
